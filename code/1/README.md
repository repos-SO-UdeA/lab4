# Ejemplo monoarchivo

## Comandos de compilacion

### Forma 1. Compilando y enlazando por pasos

```
gcc -c -Wall main.c
gcc main.o -o admin_clase.out
```

### Forma 2. Compilando y enlazando en un solo comando

```
gcc -Wall main.c -o admin_clase.out
```
